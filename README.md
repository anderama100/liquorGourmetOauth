# lIQUOR GOURMET

This application creates catalogs of spirits beverages. You can create, modify, delete and save it at any time.
   
# Technologies Used
For this project, technologies implemented are:

[x] HTML
[x] JavaScript
[x] CSS 
[x] Mongoose
[x] MongoDb
[x] Bcryptjs
[x] Heroku
